# Pizza Sales Analytics & MIS Dashboard

## Problem Statement
A restaurant chain lacked consolidated visibility into revenue trends, 
product performance, and seasonal demand patterns across a 
high-volume transaction dataset.

## Tools Used
Microsoft Excel | Pivot Tables | VLOOKUP/XLOOKUP | Power Query | Data Visualization

## Dataset
- 48,000+ pizza order records
- Data across multiple categories, sizes, and time periods

## Key Findings
- Tracked revenue by category, size, weekday, and month
- Identified peak-demand windows and seasonal sales cycles
- Surfaced top-revenue SKUs and underperforming categories

## Business Impact
- Automated KPI aggregation reduced manual reporting effort
- Weekly trend comparisons enabled faster product decisions
- Insights supported inventory optimization and campaign targeting
